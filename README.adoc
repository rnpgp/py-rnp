= Python RNP bindings

image:https://github.com/rnpgp/py-rnp/actions/workflows/ci.yml/badge.svg["Build Status", link="https://github.com/rnpgp/py-rnp/actions/workflows/ci.yml"]

This extension provides Python bindings to the
https://github.com/rnpgp/rnp[librnp OpenPGP library].

== Requirements

This extension requires Python 3.10 or newer and
https://github.com/rnpgp/rnp[librnp] 0.17.1 or newer.

librnp 0.18.1 or newer is strongly recommended: earlier releases are affected
by https://nvd.nist.gov/vuln/detail/CVE-2025-13470[CVE-2025-13470], which
caused PKESK session keys to be generated without cryptographically random
values.
